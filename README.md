# codeceptjs-demo
[Codecptjs](https://codecept.io) POC 

# Prerequisites 

- Node.js 17+

- npm

# 1. Setup:

```bash
npm install
```

# 2. Tests execution

```bash
npm test
```

# 3. UI App

```bash
npm run ui
```

# 4. Headless tests execution

```bash
npm run headless
```

# 5. [Configuration](./codecept.conf.js)
